## 2.快速开始
环境要求
1. python3.6以上
2. pip
3. 无需GPU 

### 下载代码
```bash
git clone https://github.com/sunchongwei/ResNet-Web-API.git
```
### 安装依赖
```bash 
pip install -r requirements.txt
```
